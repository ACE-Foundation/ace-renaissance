# ACE Renaissance

ACE Renaissance is the public narrative and strategic framework of the Adaptive Carbon Engine.  
It translates ACE’s institutional architecture into a clear societal vision, connecting ontology, protocol and governance to real‑world transformation.

---

## Purpose

ACE Renaissance is the human-facing layer of ACE.  
It explains:

- why ACE exists  
- how the institutional architecture fits together  
- how the ontology, methodology and protocol enable real-world climate impact  
- how ACE supports transparent, scalable and verifiable agricultural transformation  

Where ACE Foundation defines the standard, and ACE Protocol defines the data layer,  
ACE Renaissance provides the narrative that makes the system understandable and meaningful.

---

## Scope

This repository contains:

- public narrative and conceptual framing  
- societal context and strategic positioning  
- explanatory materials for non-technical audiences  
- diagrams, visual models and communication assets  
- content intended for publication on **acefoundation.net**

This repository does **not** contain:

- ontology  
- methodology  
- certificate schema  
- protocol specifications  
- engine code or algorithms  

Those are maintained in:

- **ACE Foundation** — institutional standard  
- **ACE Protocol** — constitutional data specification  

---

## Relationship to ACE Foundation

ACE Renaissance complements the ACE Foundation by providing the narrative layer.  
It does not define standards or governance; it explains them.

For the institutional standard, see:  
https://github.com/ACE-Foundation/ace-foundation

For the ACE Protocol (ACE‑CERT schema), see:  
https://github.com/ACE-Foundation/ace-protocol
